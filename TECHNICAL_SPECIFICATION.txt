Техническое задание:
Разработать систему ведения семейного бюджета

Функциональные требования:
1.	Добавлять/Удалять доход или расход по выбранной статье доходов или расходов.
2.	Справочник статей доходов, с возможностью добавления новой статьи доходов. 
3.	Справочник статей расходов, с возможностью добавления новой статьи расходов. 
4.	Справочник статей доходов, с возможностью закрытия статьи (Исчезает из выбора статей доходов, при занесении дохода) с указанной даты. 
5.	Справочник статей расходов, с возможностью закрытия статьи (Исчезает из выбора статей расходов, при занесении расхода) с указанной даты. 
6.	Предусмотреть авторизацию в системе. (Права доступа разграничивать не требуется, всегда полный набор прав)
7.	Раздел «информация о системе» с контактными данными разработчика.
8.	Формирование отчета.
	a.	Название отчета: «Сводный отчет за период»
	b.	Входные параметры отчета: «Дата с», «Дата по»
	c.	Формат строки отчета: «Дата и время операции» «Статья» «Сумма»
	d.	Сортировка по возрастанию по полю «Дата и время». (Первая строка отчета содержит самую раннюю дату)
	e.	Требования к оформлению: Фон строк для расходных статей – красный. Фон строк для доходных статей – зеленый.

Технические требования
1.	СУБД: MySQL или MSSQL
2.	Технология реализации: HTML, CSS, JQuery, AJAX и др.

Результатом является архив, содержащий:
a.	Рабочее приложение или web страница
b.	Физическая схема БД 
c.	Исходный код
d.	Пользовательская инструкция по работе с системой
e.	Инструкция по запуску системы (если требуется установка)

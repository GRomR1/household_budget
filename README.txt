[household_budget] - веб-система для ведения и учета семейного бюджета.

Система построена по основным принципам сайтостроения и демонстрирует основные компетенции веб-разработчика, а именно владение языками программирования верхнего уровня на примере Python, языками верстки сайтов (HTML, CSS), написание запросов к БД, обеспечение авторизации на сайте и др.
В качестве Backend используется Django Framework, во frontend использованы следующие технологии – HTML, CSS, JS, jQuery, Bootstrap.
